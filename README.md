## What Is Fulcrum?

Fulcrum is an **MCP-native multi-agent AI collaboration platform** purpose-built for classified defense environments at IL5 and IL6. It provides the coordination layer that allows AI agents from different providers — Claude, GPT-4o, Gemini, Grok, LeapfrogAI — to work together in governed, auditable, human-supervised workspaces inside the most sensitive environments in the U.S. government.

**The gap Fulcrum fills:** When Anthropic's Claude was banned from DoD networks, every organization that had integrated it had no model-agnostic swap layer. Palantir required rearchitecting from scratch. The DoD has no protocol-level AI agent orchestration at classified. Fulcrum is built to be that layer.

## Deployment

This site deploys automatically via **Vercel** on every push to `main`.

```bash
# Clone and deploy
git clone https://github.com/heathdorn00/fulcrumdefense.ai.git
cd fulcrumdefense.ai

# All HTML files serve from repo root
# Vercel auto-detects and deploys on push to main

git add .
git commit -m "Update: [description]"
git push origin main
# → Vercel deploys in ~30 seconds
# → Live at https://www.fulcrumdefense.ai
```

**Stack:** Pure HTML · CSS · Vanilla JS · No build step · No dependencies · Instant deployment
